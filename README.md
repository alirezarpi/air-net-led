# air-net-led
**It's simple..**

`$ chmod +x air-net-led.sh && mv air-net-led.sh /usr/bin/`

Remember to run it as **root user** (`$ sudo air-net-led.sh &` )

You can do `sudo mv air-net-led.service /etc/systemd/system/ && sudo systemctl enable air-net-led.service`
to run at boot time.

Hope you enjoy
